# 🚀 Code Factory

Code Factory is an autonomous AI-powered software engineering platform that transforms a simple app idea into a complete software project. It uses a multi-agent workflow to plan, generate code, create documentation, run tests, detect issues, and prepare the project for deployment through an interactive dashboard.

### ✨ Features

* 🤖 Multi-Agent AI Workflow
* 💡 One-Sentence App Generation
* 🏗️ Automatic Project Planning
* 💻 Code Generation
* 🧪 Automated Testing
* 🐞 Intelligent Bug Detection
* 📄 Documentation Generation
* 📊 Real-Time Progress Dashboard
* ⚡ Live Activity Logs
* 🚀 Deployment-Ready Workflow

**Tech Stack:** Python, Streamlit, AI APIs, AsyncIO, SQLite, Rich
